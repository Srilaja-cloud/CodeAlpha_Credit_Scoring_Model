# Credit Scoring Model

## 📌 Project Overview

This project develops a Credit Scoring Model using Machine Learning to predict an individual's creditworthiness based on financial information. The model helps assess whether a customer is likely to be creditworthy by analyzing factors such as income, debt, credit history, and age.

## 🎯 Objective

To build a classification model that predicts creditworthiness using historical financial data and evaluates performance using standard machine learning metrics.

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Google Colab

## 📂 Dataset Features

The dataset includes the following features:

* Income
* Debt
* Credit History
* Age

### Target Variable

* Creditworthy (1 = Yes, 0 = No)

## 🔄 Project Workflow

1. Load and prepare the dataset
2. Perform feature selection
3. Split data into training and testing sets
4. Train a Logistic Regression model
5. Generate predictions
6. Evaluate model performance
7. Visualize results using a Confusion Matrix
8. Predict creditworthiness for new customers

## 🤖 Machine Learning Algorithm

* Logistic Regression

## 📊 Evaluation Metrics

The model performance is evaluated using:

* Accuracy
* Precision
* Recall
* F1-Score
* ROC-AUC Score
* Confusion Matrix

## ▶️ How to Run

1. Open the notebook in Google Colab.
2. Run all cells sequentially.
3. Review the evaluation metrics and confusion matrix.
4. Test the model using sample customer data.

## 📁 Files Included

## 📈 Results

The model successfully classifies customers based on financial attributes and provides performance metrics to measure prediction quality.

## ✅ Conclusion

This project demonstrates how Machine Learning can be applied in the financial sector to assess credit risk and support lending decisions. The Credit Scoring Model provides an efficient approach for predicting customer creditworthiness using classification techniques.
